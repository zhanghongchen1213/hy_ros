# 参考资料

整理了本项目开发过程中参考的重要文档、开源项目及工具链接，供后续开发与维护查阅。

- **ROS 2 Documentation (Humble)**
  - 官方权威文档，包含安装、教程、API 说明。
  - 链接：<https://docs.ros.org/en/humble/>
- **ROS 2 Design**

  - ROS 2 的设计理念与架构说明（REP）。
  - 链接：<https://design.ros2.org/>

- **Ultralytics YOLOv8**
  - 实时目标检测、分割、姿态估计框架。
  - 链接：<https://docs.ultralytics.com/>
- **RKNN-Toolkit2**
  - 瑞芯微 NPU 模型转换与推理工具链。
  - 链接：<https://github.com/airockchip/rknn-toolkit2>
- **OpenCV**

  - 开源计算机视觉库文档。
  - 链接：<https://docs.opencv.org/4.x/>

- **Docker Documentation**
  - 容器化部署官方指南。
  - 链接：<https://docs.docker.com/>
- **NVIDIA Container Toolkit**

  - Docker 调用 GPU 配置指南。
  - 链接：<https://docs.nvidia.com/datacenter/cloud-native/container-toolkit/latest/index.html>

- **Colcon**
  - ROS 2 构建工具使用手册。
  - 链接：<https://colcon.readthedocs.io/>
- **Foxglove Studio**

  - 可视化调试工具文档。
  - 链接：<https://foxglove.dev/docs>
