# 产品功能

## 语音交互

1. 离线关键词、命令词识别，实现语音控制运动
2. 支持自定义关键词、命令词，实现个性化语音控制
3. 支持 sherpa-onnx ASR，Qwen3-VL-2B LLM、百度大模型音色复刻 TTS
4. 全流式语音交互，无需等待，实时响应
5. 支持多模态输入，可同时识别语音、文本、图像
6. 支持多音色管理，可自定义音色，实现个性化语音交互

## 运动控制

1. PID 速度环、位置环、角度环控制，实现精准运动
2. 舵机控制，实现精准运动
3. Web 端远程控制，实时数字孪生效果

## 视觉交互

1. yolov8n 黄油目标检测，手掌目标检测及自主跟随
2. Web 端实时显示摄像头画面以及检测结果

## 定位和导航

1. 支持加载第五和 ACML 定位算法
2. 支持基于定位的自主导航
3. 支持 DWA 或 TEB 路径规划算法
